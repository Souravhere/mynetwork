## 🚀 Installation

```bash
git clone https://github.com/yourusername/mynetwork.git
cd mynetwork
npm install
```

Make sure you have `speedtest-cli` installed:

```bash
npm install -g speedtest-cli
```

---

## 💻 Usage

```bash
npm start
```

or

```bash
node bin/cli.js
```
